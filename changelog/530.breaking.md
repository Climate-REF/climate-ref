Changed `get_branding_suffix`, `get_realm`, and `get_cmip7_compound_name` to require a `table_id` parameter in addition to `variable_id`, enabling Data Request compound name lookups. Renamed `_get_dreq_entry` to `get_dreq_entry` as a public API. The `convert_cmip6_to_cmip7_attrs` function now derives the `region` attribute from the Data Request entry instead of defaulting to `"glb"`.
