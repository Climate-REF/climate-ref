ILAMB metric values now carry the shared `kind` field, distinguishing reference (observation) series from model series without the previous `"Reference"` sentinel.
Series units are presented in a clean CF form so a model series and its reference agree, the typed presentation fields (`value_units`, `value_long_name`, `index_units`, `calendar`) are populated, and comparison scalars record the reference they were scored against so they can be grouped by it.
