Added an opt-in mechanism to fabricate extended CMIP7 `historical` test coverage past 2014 (`shift_time_axis_end`, `CMIP7Request(extend_historical_to=...)`, and a `generate_cmip7_catalog.py` option), so diagnostics with post-2014 CMIP7 windows can be solved and minted. Off by default, so existing CMIP7 conversions and baselines are unchanged.
