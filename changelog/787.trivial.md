Added an opt-in mechanism to fabricate extended CMIP7 `historical` test coverage past 2014 (`shift_time_axis_end`, `CMIP7Request(extend_historical_to=...)`, and a `generate_cmip7_catalog.py` option), so diagnostics with post-2014 CMIP7 windows (e.g. climate-drivers-for-fire, 2002-2021) can be solved and minted. The time-axis shift is off by default, so existing CMIP7 conversions and minted baselines are unchanged. The regenerated solve-test catalog extends fabricated CMIP7 `historical` coverage to 2021, which updates the CMIP7 recipe timerange for the ENSO diagnostics (which use the full historical span).
