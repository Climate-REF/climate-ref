Made `long_name` nullable on the obs4MIPs, obs4REF and PMP climatology dataset tables.
It is read from the variable's attributes, which obs4MIPs does not require,
and the published obs4REF FLUXNET2015 `gpp` dataset omits it.
The `NOT NULL` constraint turned that single missing attribute into an `IntegrityError`
that aborted the entire obs4MIPs ingest.
Every other dataset table already treated `long_name` as optional.
